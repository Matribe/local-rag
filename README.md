


## Installation

Il faut installer ollama : https://ollama.com/


Puis installer Mistral via : 

```bash
ollama pull mistral
```


Vérifier que le serveur ollama est bian lancé via : 

```bash
ollama serve
```

Pour lancer le projet :  

```bash
# streamlit run main.py
python3 main.py
```
